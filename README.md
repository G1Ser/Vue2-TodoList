# Vue2-TodoList
本项目是基于vue2、vueX技术开发的TodoList界面，通过本次项目，详细了解vue技术的应用。
## 1.项目说明
### 1.1.技术栈
|   说明  |    技术栈     |
|:--------:|:----------:|
| Js框架 |    Vue2   |
| 状态管理 |    VueX   |
| 脚手架   |    Vue CLI    |
| 本次存储 | Localstorage |
